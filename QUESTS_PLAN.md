## MODS À AJOUTER

### Extra Utilities 2 — Priorité HAUTE
```
Pourquoi : E2E l'utilise dans 40+ recettes. Contenu unique irremplaçable.

Items clés :
- Angel Block (placer des blocs en l'air — CRUCIAL en compact machine)
- Cursed Earth (mob spawn x10 — mob farm OP)
- Resonator (Grid Power system)
- Speed Upgrade (accélère machines)
- Golden Lasso (capture mobs)
- Trash Can (items, fluids, energy)
- Generators variés (Furnace, Culinary, Ender, etc.)
- Sun Crystal (composant craft)
- Quantum Quarry (mining dimensionnel)
- Chunk Loading Ward

Âge d'introduction : progressif
  Âge 0 : Angel Blocks (SUPER utile en compact machine)
  Âge 1 : Trash Cans, Golden Lasso
  Âge 3 : Generators, Speed Upgrades, Resonator
  Âge 4 : Cursed Earth, Quantum Quarry
  
Quêtes ajoutées : ~20-25 réparties sur les âges
```

### Dark Utilities — Priorité MOYENNE
```
Pourquoi : Vector Plates et mob traps complètent les mob farms.

Items clés :
- Vector Plates (poussent les mobs/items dans une direction)
- Mob Traps (tuent les mobs automatiquement)
- Ender Tether (chunk loading basique)
- Filter (tri d'items basique)
- Snare (capture mobs)

Âge d'introduction :
  Âge 1 : Vector Plates (utile en compact pour pousser items)
  Âge 3 : Mob Traps, Snare

Quêtes ajoutées : ~5-8
```
